LIIM Centre কী?

Library Innovation and Information Management Centre (LIIM Centre) হলো চট্টগ্রাম বিশ্ববিদ্যালয়ের দর্শন বিভাগের সেমিনার লাইব্রেরিতে অবস্থিত একটি স্বতন্ত্র ডিজিটাল উদ্যোগ।

এটি কোনো বিভাগীয় প্রশাসনিক কাঠামোর অংশ নয় — বরং এটি একজন উদ্যমী লাইব্রেরি পেশাদারের ব্যক্তিগত উদ্যোগে গড়ে ওঠা একটি আধুনিক তথ্য ও জ্ঞান ব্যবস্থাপনা কেন্দ্র।

ডিজিটাল ক্যাটালগ, গবেষণা সহায়তা, তথ্য-প্রযুক্তি প্রশিক্ষণ এবং উন্মুক্ত জ্ঞানভান্ডার গড়ে তোলাই এই কেন্দ্রের মূল কাজ।

  • পদ্ধতি ১ — PDF ড্র্যাগ ও ড্রপ: পিডিএফ ফাইল সরাসরি Mendeley-তে টেনে আনুন। সফটওয়্যার স্বয়ংক্রিয়ভাবে লেখক, শিরোনাম ও বছর খুঁজে নেবে।
  • পদ্ধতি ২ — DOI দিয়ে: File → Add Entry Manually → DOI বক্সে পেপারের DOI নম্বর পেস্ট করুন। বাকি তথ্য স্বয়ংক্রিয় আসবে।
  • পদ্ধতি ৩ — Web Importer (Browser Extension): ব্রাউজারে Mendeley Web Importer এক্সটেনশন ইনস্টল করুন। যেকোনো ওয়েবসাইট বা Google Scholar থেকে এক ক্লিকে সেভ করুন।
  • পদ্ধতি ৪ — ম্যানুয়ালি: File → Add Entry Manually → ধরন বেছে (বই/জার্নাল/থিসিস) সব তথ্য নিজে টাইপ করুন।

💡 টিপ: Google Scholar-এ কোনো পেপার খুঁজে পেলে Web Importer দিয়ে এক ক্লিকে Mendeley-তে সেভ করুন — সময় বাঁচবে।

গবেষণার বিষয় অনুযায়ী ফোল্ডার তৈরি করুন এবং ট্যাগ ব্যবহার করে দ্রুত খুঁজে পান।

  • বাম প্যানেলে "Create Folder" ক্লিক করুন
  • ফোল্ডারের নাম দিন (যেমন: থিসিস অধ্যায় ১, ইসলামী দর্শন, কান্টের নীতিশাস্ত্র)
  • রেফারেন্স টেনে ফোল্ডারে রাখুন
  • ডান প্যানেলে "Tags" বক্সে কীওয়ার্ড যোগ করুন
  • স্টার (★) চিহ্ন দিয়ে গুরুত্বপূর্ণ পেপার চিহ্নিত করুন

💡 টিপ: একটি পেপার একসাথে একাধিক ফোল্ডারে রাখা যায়। ফোল্ডারের মধ্যে সাব-ফোল্ডারও তৈরি করা যায়।

৫
📝 Word Plugin ইনস্টল ও সাইটেশন যোগ করা সবচেয়ে দরকারি

Mendeley-এর সবচেয়ে কার্যকর ফিচার হলো MS Word-এ সরাসরি সাইটেশন দেওয়া। একটি Plugin ইনস্টল করলেই হবে।

  • Mendeley Desktop খোলা অবস্থায়: Tools → Install MS Word Plugin
  • Word খুলুন — উপরের মেনুতে "References" ট্যাবে "Mendeley Cite" দেখবেন
  • থিসিস লেখার সময় যেখানে সাইটেশন দরকার সেখানে কার্সর রাখুন
  • "Insert Citation" ক্লিক করুন → পেপারের নাম বা লেখক লিখে সার্চ করুন
  • পেপার বেছে "OK" করুন → সাইটেশন স্বয়ংক্রিয়ভাবে যোগ হবে
  • থিসিসের শেষে References/Bibliography পেজে: "Insert Bibliography" ক্লিক করুন
  • সব রেফারেন্স স্বয়ংক্রিয়ভাবে নির্বাচিত স্টাইলে তালিকা তৈরি হবে
📝 MS Word — Mendeley Cite প্যানেল
References ট্যাব → Mendeley Cite → সার্চ বক্সে লেখক বা শিরোনাম → পেপার বেছে Insert

💡 টিপ: Word বন্ধ থাকলে Plugin কাজ করবে না। Mendeley Desktop ও Word — দুটোই খোলা রাখুন।

🎨 সাইটেশন স্টাইল পরিবর্তন করুন APA / MLA / Chicago

বিভিন্ন জার্নাল ও বিশ্ববিদ্যালয় ভিন্ন সাইটেশন স্টাইল চায়। Mendeley-তে ৯,০০০+ স্টাইল আছে — যেকোনোটি এক ক্লিকে পরিবর্তন করা যায়।

স্টাইলকোথায় ব্যবহারউদাহরণ
APA 7thসামাজিক বিজ্ঞান, মনোবিজ্ঞান, দর্শনKant, I. (1785). Groundwork.
MLA 9thমানবিক বিজ্ঞান, সাহিত্যKant, Immanuel. Groundwork. 1785.
Chicagoইতিহাস, ধর্মতত্ত্বKant, Immanuel. 1785. Groundwork.
IEEEপ্রকৌশল, প্রযুক্তি[1] I. Kant, Groundwork, 1785.
  • Word-এ Mendeley Cite প্যানেলে "Citation Style" বোতাম ক্লিক করুন
  • সার্চ বক্সে APA / MLA / Chicago লিখুন
  • পছন্দের স্টাইল বেছে "Update" করুন
  • পুরো ডকুমেন্টের সব সাইটেশন ও Bibliography স্বয়ংক্রিয় আপডেট হবে

💡 দর্শন বিভাগের জন্য: সাধারণত APA 7th বা Chicago Author-Date স্টাইল ব্যবহৃত হয়। জার্নালের গাইডলাইন চেক করুন।

👥 গ্রুপ লাইব্রেরি — দলগত গবেষণা

❓ সাধারণ প্রশ্নোত্তর

PDF স্বয়ংক্রিয়ভাবে তথ্য নিচ্ছে না?

পেপারটি সিলেক্ট করে ডান ক্লিক → "Update Details" করুন। অথবা DOI ম্যানুয়ালি দিন।

Word-এ Plugin দেখা যাচ্ছে না?

Mendeley Desktop থেকে Tools → Install MS Word Plugin আবার করুন। Word রিস্টার্ট দিন।

বাংলা লেখক নাম কীভাবে দেব?

ম্যানুয়ালি এন্ট্রি করুন। Author বক্সে বাংলায় নাম লিখুন। প্রতিটি লেখকের পর Enter চাপুন।

Zotero-র সাথে পার্থক্য কী?

Mendeley Elsevier-এর (বেশি PDF সংগঠন সুবিধা), Zotero ওপেন সোর্স (বেশি ওয়েব ইম্পোর্ট সুবিধা)। দুটোই ভালো।
